# NES Emulator
原先只是一個因興趣而做的side project，但誤打誤撞變成專題的NES(紅白機)模擬器。

## Getting Started
Prerequisites: CMake, C++ compiler, OpenGL, OpenAL
```bash
cmake -S . -B build
cmake --build build --config Release
```
